#IT Company – Business Website#

This project represents a business website created using semantic HTML structure and organized layout sections.
The page includes a header navigation menu, hero section, about section, services presentation, portfolio showcase,
and a contact section with an embedded Google Maps iframe and a contact form.
The layout uses reusable containers and utility classes for alignment and spacing,
while images, icons, and structured content blocks are used to present services and projects in a clear and professional way.
The project also demonstrates the use of a CSS reset (Normalize.css), custom fonts, 
and organized component-based sections which are common practices in professional front-end development.
